follow()

static follow(filePath, options): AsyncGenerator<SessionLogEntry, void>

Defined in: src/session/session_log_tailer.ts:42 

Follow filePath, yielding each newly-appended SessionLogEntry until the run is done (then one final drain) or the consumer aborts. A missing file is treated as empty and polled until it appears. Malformed/torn lines are skipped, exactly like SessionLogReader.readAll.
