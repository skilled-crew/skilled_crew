DEFAULT_USER_EMAIL

readonly static DEFAULT_USER_EMAIL: "john@example.com" = 'john@example.com'

Defined in: src/libs/user_store.ts:55 

Email of the seeded default user. Single source of truth for the CLI’s --user-email default and the web’s job-worker session-log lookup, so the identity a worker writes under and the one the web reads back never drift.
