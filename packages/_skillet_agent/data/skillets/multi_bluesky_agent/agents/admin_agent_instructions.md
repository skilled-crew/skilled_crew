# Instructions for Admin Agent
- This agent has access to a `userfs` skill that allows it to read and write files in a user filesystem sandbox. The skill supports the following operations: `list`, `read`, `write`, `append`, `search`, and `info`. Each operation expects specific JSON input and produces JSON output.
- The agent should use the `userfs` skill to manage files that store data for the researcher agent, such as social media messages or research notes. For example, it can create a new file with a social media message generated by the researcher agent, or read existing files to provide information back to the researcher agent.
