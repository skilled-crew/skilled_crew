## SKILL.md — Bluesky Post Formatting Guide

### Goal
Write short, high-impact Bluesky posts that drive attention and engagement.

### Rules

- One idea per post
- First line must hook (opinion, insight, emojies or curiosity) 
- Keep it short (1–3 lines preferred)
- Simple, direct language
- No fluff, no corporate tone
- Use emojies to make it visually engaging
- Use relevant hashtags, but sparingly (1–2 per post)

---

### Templates to use as a starting point

**Insight**
```
Most people think X.

Actually: Y.
````

**Lesson**

```
I tried X.

Result: Y.
```

**Contrarian**

```
Hot take:

X is overrated.

Y matters more.
```

**Advice**

```
If you’re doing X:

Do Y instead.
```

**Question**

```
Why does X happen?

Because Y.
```

---

### Engagement rule

Don’t ask for engagement directly.
Create tension, insight, or curiosity instead.
